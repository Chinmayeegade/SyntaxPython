PRINT FUNCTION

Using the print function, the required characters can be outputted.

Requisites:

1) Built in print statement (*Error will be generated if any letter in 'print' is capitalized.)

2) A pair of parentheses () adjacent to the statement (*No spaces between 'print' and parentheses.)

3) Single or double inverted commas ' '/" "ONLY if a string is to be printed. (*Using inverted commas to print numeral(s) will convert the numeral(s) to a string.)

INPUT:

To be entered in the input console in an IDE (Integrated Development Environment) of a Python compiler)

print("Hello there!!")

print(25)

OUTPUT:

Will be generated by pressing the 'run' key (triangle shape)

The text/numerals entered within the print function shall be generated.

Hello There!!
25

Process finished with exit code 0
