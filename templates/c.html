input FUNCTION

Using the input function, characters can be inputted from the user for further use in the program.

Requisites:

1) Built in input statement (*Error will be generated if any letter in 'input' is capitalized.)

2) A pair of parentheses () adjacent to the statement (*No spaces between 'input' and parentheses.)

Tips:

1) Try to make the input sound as if you are communicating with the user.

2) Preferably add a punctuation and a space tab between the text in the input function, and the closing inverted commas.

**CONCATENATION IS NOT POSSIBLE WITHIN AN 'INPUT' STATEMENT.

INPUT:

input("Please enter your name: ")

OUTPUT:

The text entered within the input function will be printed along with user accessibility to enter the input.

Please enter your name: David

Process finished with exit code 0

**'David' is the input entered by the user
