introduction to strings

In Python, a string is a literal constant or a group of characters.

A string can comprise of letters, numbers, spaces and as well as special characters.

While in a function, strings must be enclosed by single or double inverted commas.

Strings are immutable, that is they cannot be changed once specified.

Strings can be formatted, added, multiplied and sliced (*Refer respective posts)

*Numeric values can be converted and stored into strings by using the built-in string function: str() .

**Strings can be converted completely into upper or lower or titlecases using the .upper() or .lower() or .title() functions, respectively, just after the closing inverted commas of the string.

INPUT:

To be entered in the input console in an IDE (Integrated Development Environment) of a Python compiler)

print("How Are You Doing Today?".upper())

print("How Are You Doing Today?".lower())

The output will be empty as there is not print command in the input, however 5 will be saved as a string.

OUTPUT:

Will be generated by pressing the 'run' key (triangle shape)

The text/numerals entered within the print function shall be generated.

HOW ARE YOU DOING TODAY?
