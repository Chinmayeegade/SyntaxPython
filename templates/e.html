concatenation

Concatenation the process of putting together strings, numbers and variables using either a plus sign + or a comma , .

Requisites:

1) Print function (print statement and parentheses ())

2) 2 or more strings, both enclosed in inverted commas ' '/" " and separated by a plus sign or a comma +/, .

3) Single or double inverted commas ' '/" "ONLY if a string is to be printed. (*Using inverted commas to print numeral(s) will convert the numeral(s) to a string.)

4) ALL NUMERIC VALUES THAT ARE TO BE CONCATENATED MUST BE CONVERTED TO STRINGS USIN THE STRING FUNCTION str()

Tips:

1) A space tab between two different strings will be auto-generated in the output if a comma is used, but will have to be manually added if a plus sign is used.

**CONCATENATION IS NOT POSSIBLE WITHIN AN 'INPUT' FUNCTION.

INPUT:

a = 3678 / 6
b = 87
c = a * b
print("The answer is " + str(c) + ". Hence, there are", c, "units.")

OUTPUT:

The answer is 53331.0. Hence, there are 53331.0 units.
