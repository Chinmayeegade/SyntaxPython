VARibles and data types

Python has 5 fundamental forms in which data can be stored. These are known as data types. They are: Strings, Numbers, Lists, Tuples and Dictionaries.

Variables are ‘objects’ used to identify and store a specific data type.

Variables play an utmost important role in programing.

A single variable can store and reproduce an infinite amount of data as long as the data type is constant.

Data is stored in a variable by defining the variable, followed by an equal sign = , followed by the data.

A variable, once defined can be inputted innumerable times in the respective program instead of tediously inputting the whole data each and every time.

A variable can also be changed/modified mid-program using the equal sign = .

When any variable is to be printed (using the print function), the value(s) stored within that variable shall be outputted.

Rules for assigning variables:

1) Variables are case-sensitive. (That is, num1 and Num1 are considered as two different variables)

2) Variables must begin with an underscore _ or a letter (upper or lower case).

3) A variable may not contain any space tabs.

4) Numbers, letters (upper and lower cases) and underscores are allowed in variables, however the variable should not begin with a number.

5) Special characters are forbidden within a variable.

6) While stating or reproducing a variable within a program, it should NOT be enclosed within inverted commas.

INPUT:

My_variable5 = "Great minds think alike."

print(My_variable5)

OUTPUT:

Great minds think alike.
