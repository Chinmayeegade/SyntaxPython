STRING MULTIPLICATION

Using string multiplication, the same string can be printed a number of times.

Requisites:

1) Print function (print statement and parentheses ()) .

2) A string enclosed in inverted commas “ ”/ ‘ ’ .

3) An Asterix * .

4) A number equal to the number of times the string is to be outputted.

Tips:

A space tab will have to be manually added in the program after the string, so that the strings printed in the output will be separated from one another.

INPUT:

print("Hello! " *5)

OUTPUT:

Will be generated by pressing the 'run' key (triangle shape)

The text/numerals entered within the print function shall be generated.

Hello! Hello! Hello! Hello! Hello!

Process finished with exit code 0
