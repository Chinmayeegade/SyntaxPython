FORMATTING STRINGS

Using string formats, the position of the string in the output can be specified.

Requisites:

1) Print function (print statement and parentheses ())

2) Built-in format statement (*Error will be generated if any letter in 'format' is capitalized.)

3) 2 strings, both enclosed in inverted commas ' '/" " and separate by a comma , :

The first string should comprise the text/numeral that is to be outputted.

The second string should include the directed alignment of the text/numeral along with the field width.

For example,

~ Central alignment in a width of 20 units: “^20”

~ Right justified in a width of 10 units: “>10”

~ Left justified in a width of 50 units: “<50”

INPUT:

print(format("Syntax_Python @ 123", " ^30"))

print(25)

OUTPUT:

The first text/numerals in the first string shall be outputted, centrally aligned in width range of 30 units.

Syntax_Python@123
