SLicing methods

Using slicing, only a specified number of characters in the data type can be outputted.

SLICING METHODS CAN ONLY BE APPLIED TO STRINGS AND LISTS

Requisites:

1) A string saved as a variable (*Refer post on variables and data types.)

2) Print function (print statement and parentheses ()) .

3) A square bracket [] .

4) Index number(s) of the character(s) and (if necessary) a colon : .

~ A single number inside the square bracket will print the string character having index number equal to that number.

~A number inside the square bracket followed by a colon will print the character having index number equal to that number along with all the characters that succeed it.

~A colon inside the square bracket followed by a number will print the characters that precede the character having index number equal to that number. (*The character having index number equal to that number will not be printed.)

~ Two numbers separated by a colon inside the square bracket will give an output that prints the character having index number equal to the first number in the square bracket along with characters having index numbers equal to the numbers that lie between the 2 numbers.

~ Two colons inside the square bracket followed by a number (n) will print the 0th character along with every nth character in the string

~ Two numbers (x & y) separated by two colons will output the xth character, followed by every yth character in the string.

Tips:

Index numbers of characters in Python start from 0, not 1.

Spaces are considered as characters.

A number inserted inside the square bracket with a negative sign will output the character in the string having index number equal to that number, considering the string in the reverse order.

INPUT:

sentence = ("Better late than never.")
print(sentence[8])
print(sentence[5:])
print(sentence[:2])
print(sentence[5:8])
print(sentence[::4]
print(sentence[7::2])
