NUMERIC CONSTANTS

Numbers, in Python, are literal constants which refer to specific numeric values, and hence are immutable.

1) Integers: Comprises whole numbers and negative integers. (ex. -45, 8736)

2) Floating Numbers: Comprises decimal numbers. (ex. 4.675, 0.86)

3) Complex numbers: Comprises imaginary numbers. (ex. 10j, 2+3j) (*j is the only valid imaginary value in Python, and it must be preceded by a constant number)

Commas CANNOT be used in numeric values.

While programming with floating numbers, arithmetic overflow of arithmetic underflow errors may occur if the number is infinitesimally large or small, respectively.

Numbers, unlike strings should not be enclosed in inverted commas, and will be considered as strings if they are.

Python has built in (to be used within the 'print' function) statements to convert any numeric value into any of the three number types:

1) Integers: int() statement (*Any decimal number will be rounded DOWN to the nearest whole number.)

2) Floating Numbers: float() statement (*Any whole number will be converted and stored as a decimal number.)

3) Complex Numbers: complex() statement (*Any whole or decimal number will br converted and stored a a sum of a whole/decimal number and an imaginary number )

INPUT:

print(int(3.7))

print(float(5))

print(complex(7.8))

OUTPUT:

3

5.0

(7.8 + 0j)
