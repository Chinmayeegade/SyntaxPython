NUMERIC OPERATIONS

Python has a number of built-in as well as callable functions to perform various mathematical operations.

1) Addition

Two numbers can be added by simply using the plus sign + .

INPUT:

print(3+5)

OUTPUT:

8

Process finished with exit code 0

2) Subtraction:

Two numbers can be subtracted by simply using the minus sign - .

INPUT:

print(30-37)

OUTPUT:

-7

Process finished with exit code 0

3) Multiplication:

Two numbers can be subtracted by simply using the asterisk * .

INPUT:

print(30*7)

OUTPUT:

210

Process finished with exit code 0

4) Division

Two numbers can be divided by simply using a front slash / and the answer will be obtained in a decimal form.

If two numbers with 2 front slashes // between them are inputted inside a print function, only a whole number quotient (rounded DOWN) of the division problem will be outputted.

If two numbers with a percentage sign % between them are inputted inside a print function, the remainder of the division problem will be outputted. (Modulus function)

INPUT:

print(10 / 2)

print(17 // 4)

print(80 % 6)

OUTPUT:

5



4



2

Process finished with exit code 0

5) Power

A number can be raised to the power of a second number using two methods:

1) Inputting two numbers and 2 asterisks ** between them inside a print function.

2) Using the power function pow() inside a print function and entering two numbers inside the parentheses, separated by a comma.

In both cases, the fist number will be raised to the power of the second number and the answer will be outputted.

If the power function pow() is given 3 values, separated by commas, the modulus of the first number raised to the power of the second number with respect to the third number will be the answer.

INPUT:

print(2 ** 8)

print(pow(2, 8))

print(pow(2, 8, 7))

OUTPUT:

256

256

4

Process finished with exit code 0

(**If more than two of the above operations are put together in a single equation, BODMAS rules will the solved to output the answer.)

6) Maximum and Minimum:

Inputting a series of numbers (separated by commas) in the maximum function max() and printing it will output the largest number in the series.

Inputting a series of numbers (separated by commas) in the minimum function min() and printing it will output the smallest number in the series.

INPUT:

print(max(34, 87, 75, 9876, 4))

print(min(3683, 8937, 6, 837, ))

OUTPUT:

9876

6

Process finished with exit code 0

7) Round Function:

A floating number printed within the round function round() will be rounded to the nearest whole number.

INPUT:

print(round(3.7673))

OUTPUT:

4

Process finished with exit code 0

7) Binary Function

A whole number printed within the binary function bin() will output the binary form of that number.

INPUT:

print(bin(56))

OUTPUT:

0b111000

Process finished with exit code 0

8) Octal Function:

A whole number printed within the octal function oct() will output the octal form of that number.

INPUT:

print(oct(67))

OUTPUT:

0o103

Process finished with exit code 0

9) Hexadecimal Function:

A whole number printed within the hexadecimal function hex() will output the hexadecimal form of that number

INPUT:

print(hex(6))

OUTPUT:

0x6

Process finished with exit code 0
