LISTS

Lists are a form of data types in Python, and a single list can store a varied amount strings, variables, Boolean values and numeric values/operations, lists, tuples, dictionaries.

Lists must be enclosed in a pair of square brackets [] and are stored as variables.

Each element within a list must be separated by a comma , .

Lists are mutable and hence, can be changed/modified mid-program.

The index number of the first element in a list is 0.

A list, once changed/modified will be the new data type saved under that variable (not the original).

**A tuple can be converted into a list by using the list function list() and typing the variable of the tuple in the parentheses.

INPUT:

Sum1 = 4 + 7

Sum2 = 78 + 67

Sum3 = 654 + 434

Numbers = [2, 76, 52, 98, 6, 854, 6]

Strings = ["a", "b", "c", "d", "e"]

Boolean_values = [True, False]

Operations = [Sum1, Sum2]

Colors = ["red", "green", "blue"]

# Print the list of colors

print(Colors)

# Print the elements of list ‘numbers’ having index numbers from 1 to 4
print(Numbers[1:5])

# Print ‘operations’ list 4 times

print(Operations * 4)

# Concatenate 2 lists

print(Colors + Boolean_values)

# Copy the list of strings and save it under the variable ‘letters’

letters = Strings.copy()

print(letters)

# Print the largest number in the list of numbers

print(max(Numbers))

# Append (add at last index number) the color ‘yellow’ to the list of colors and then print it

Colors.append("Yellow")

print(Colors)

# Print the list of strings in the reverse order

Strings.reverse()

print(Strings)

# Print the list of numbers in ascending order

Numbers.sort()

print(Numbers)

# Insert Sum3 to the list of Operations at index number 1 and then print it

Operations.insert(1, Sum3)

print(Operations)

# Remove the element at index number 3 in the list of Strings and then print it

Strings.pop(3)

print(Strings)

# Remove the letter ‘a’ in the list of Strings and then print it

Strings.remove("a")

print(Strings)

# Clear/Empty the list of boolean values and then print it

Boolean_values.clear()

print(Boolean_values)

# Add 'k' to the list of letters

letters.extend("k")

print(letters)

#Print the number of times the number '6' appears in the list of numbers
print(Numbers.count(6))

#Create a list of all the above stated lists and then print it
final_list = [Numbers,Strings,Colors,Boolean_values,letters,Operations]
print(final_list)
