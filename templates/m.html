tuples

Tuples are a form of data types that cannot be changed (mutated) once defined.

Each element within a tuple must be separated by a comma.

A single Tuple can store a varied amount strings, variables, Boolean values and numeric values/operations, lists, tuples, dictionaries.

2 or more tuples can be concatenated.

The index number of the first element in a tuple is 0.

Trying to make any changes in a previously stated tuple will give an error.

**A list can be converted into a tuple by using the tuple function tuple() and typing the variable of the tuple in the parentheses.

INPUT:


  letters = ("a","b","c","d","e")

  numbers = (1,3,5,7,9,10+1,2+3)

  #Print the index number of element 'b' in the tuple of letters

  print(letters.index("b"))

  #Print the number of times 5 appears in the tiple of numbers

  print(numbers.count(5))

  #Print the 'minimum' and 'maximum' letter in the tuple of letter (alphabetical order)

  print(min(letters))

  print(max(letters))

  # Print the elements of tuple ‘numbers’ having index number 3

  print(letters[3])

  #Concatenate the tuples 'numbers' and 'letters'

  print(letters+numbers)

  #Create a list of tuples and print it

  list = [numbers,letters]

  print(list)
