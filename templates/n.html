dictionaries

Dictionaries are a form of data types in Python, which store data in key-value pairs.

Dictionaries must be enclosed in a pair of curly brackets {} and are stored as variables.

Each element within a dictionary comprises two data types: a key, and itâ€™s value, both of which are separated by a colon : .

Every element in a dictionary is separated from the next by a comma , .

Keys in dictionaries are immutable (They must remain constant throughout a program), however values of keys can be changed mid-program.

2 or more dictionaries cannot be concatenated.

The index number of the first element in a dictionary is 0.

2 keys can have the same value, but if two values have the same key, only the second value will be stored.
