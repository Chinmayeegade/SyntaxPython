if statement

The if statement in Python makes conditional operation and handling of data possible.

(** ‘if’ is a keyword)

The ‘if’ statement is to be succeeded by a condition(s) and a COLON : . If the condition(s) is fulfilled, the program will move on to the next instruction, which will then be executed. (THIS INSTRUCTION MUST BE ENTERED WITH AN INDENT PRECEDING IT.) If the stated condition is not fulfilled, the program will either end or escape the if statement.

Two or more conditions given to an if statement should be separated by either of the keywords and or or or in or not in .

More than one if statement can be used as long as the conditions differ.

INPUT:

a = 9*5
b = 6*8
if a > b:
    print(a,">",b)
if a == b:
    print(a,"=",b)
if a < b:
    print(a,"<",b)

OUTPUT:

45 < 48

Process finished with exit code 0
