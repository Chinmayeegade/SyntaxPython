FOR LOOP

A for loop is a structure that requires a range and a prompt, and the prompt/instruction will be continued for as long as the range is abided.

Requisites:

1) A for loop is called by using the keyword for

2) A variable that has not been previously stated throughout the program (Let's take x for example)

3) Either x in (*data type): or function range() in which the parentheses inputs 3 values in the following order(separated by commas):

a. Start value of the variable: This number is included in the prompt operations

b. End value of the variable: This number is NOT included in the prompt operations

c. The value by which the variable is to be increased (positive value) or decreased (negative value)

For loops with range function must be printed as follows: for x in range(3,9,1): (**Don’t forget the colon!) Here, x will start from 3, update itself by +1 after lopping once and will continue looping until x=9-1=8

The following line(s) will be composed of an indent and a prompt. (**While loops are usually used for definite repetitions of the prompt.) (**’and’ and ‘or’ keywords can be used between conditions)

INPUT:

list = ["A","B","C","D"]
for x in list:
    print(x)

OUTPUT:

A

B

C

D

Process finished with exit code 0
