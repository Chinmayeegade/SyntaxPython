import FUNCTION

The Import function in Python can import various saved files in Python libraries, and hence provide access to a wide range of statements, functions and classes. It is highly useful and time-saving.

There are 2 fundamentals to an import function.

1) 1. The import statement: import which is to be succeeded by the library name.

2) The 'from' statement: written as: from (library name) import (module name/symbol)

(**Library and module names are case sensitive)

There are innumerable libraries available for use, like math, sys, kivy, pygame, binacii, etc.

Tips:

1) Try to make the input sound as if you are communicating with the user.

(**There could be restrictions/downloading policies of libraries based on which IDE you use.)

INPUT:

import math
from math import *
number = int(input("Enter a number: "))
print(factorial(number)) #Factorial
print(sqrt(number)) #square root
print(log(number,10)) # log of number to base 10
