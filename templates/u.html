DEF keyword

The def keyword in python is used to call define a specific finite function that will process only when it is called, and will otherwise not interfere in the program.

The def keyword is succeeded by a variable (to name that function), a parentheses () , and a colon : .

The parentheses can also input one or more variable(s) (known as ‘argument(s)’),separated by comma(s), each of which can be concatenated in the program, within the def statement.

The part of the program placed within the ‘def’ statement will have to be appropriately indented.

The function can be called by typing the variable of the function along with a parentheses. (Only after escaping the ‘def’ function.)

While calling a function, the parentheses can input a string/number, which will automatically be equated to the stared ‘argument(s)’.

INPUT:

def my_fuction(Name,age):
    print("Hello",Name,"!!")
    year = 2021-age
    print("You were born in",year,".")

my_fuction("John", 34)

OUTPUT:

Hello John !!

You were born in 1987 .

Process finished with exit code 0
