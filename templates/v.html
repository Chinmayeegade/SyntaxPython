Lambda Function

The lambda keyword acts as a def function but does not require a command to be executed. Once stated, it can be used anywhere throughout the program.

The keyword lambda should be equated to a variable followed by the variable(s) to be operated, separated by comma(s) , and a colon : and the operation to be implemented.

To process this function, print the variable assigned to the ‘lambda’ function along with a pair of parentheses comprising of the values of the variable(s) to be operated, each separated by commas.

INPUT:

square_of_sum = lambda a, b, c, x, y, z : (a+b+c+x+y+z)*(a+b+c+x+y+z)

print(square_of_sum(2, 6, 57, 4, 23, 8))

OUTPUT:

The text entered within the input function will be printed along with user accessibility to enter the input.
