comments

Comments in python are instructions/manuals entered amidst the program for the programmer to comprehend the code.

Comments are never printed/generated in the output.

To write a comment, print a hashtag # and then type the comment.

Alternatively, comments can also be added by enclosing the text within three pairs of double inverted commas.

INPUT:

Num1 = 54

Num2 = 66

# Print the summation of Num1 and Num2

print(Num1 + Num2)

OUTPUT:

120

Comparison operators are used to compare/equate numerical values/variables.

Following are the comparison operators and their meanings:

Operator Meaning
== Equal to
!= Not Equal to
> Greater than
< Less than
>= Greater than/Equal to
<= Less than/Equal to

INPUT:

a = 9 * 7

b = 21 * 3

print(a == b)

OUTPUT:

True
